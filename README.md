# Deploy Netflix Clone on Cloud using Jenkins - DevSecOps Project!

## 📌 Project Overview

This Netflix Clone Web Application is a **DevSecOps-focused academic project** demonstrating a complete CI/CD pipeline 
with built-in security checks and cloud deployment practices.

The project integrates automated build, test, security scanning, containerization, and deployment workflows using
industry-standard DevSecOps tools and practices to simulate real-world production workflows.

## 🛠️ Tech Stack
- Frontend: HTML, CSS, JavaScript  
- Cloud Platform: AWS  
- Version Control: Git, GitHub  
- CI/CD: Jenkins (Pipeline Automation)  
- Containerization: Docker  
- Orchestration: Kubernetes  
- Build Tool: Maven  
- Static Code Analysis: SonarQube  
- Security Scanning: Trivy 

## 🚀 Features
- Responsive Netflix-style user interface  
- Automated CI/CD pipeline using Jenkins  
- Docker-based application containerization  
- Kubernetes-managed deployment and scaling  
- Continuous security scanning with Trivy  
- Static code quality analysis using SonarQube  
- Cloud deployment on AWS infrastructure  
- Version control and workflow automation via GitHub  

## 🧩 System Architecture
1. Application code hosted on GitHub  
2. Jenkins triggers the CI/CD pipeline on code changes  
3. Maven builds the project artifacts  
4. SonarQube performs static code quality checks  
5. Docker packages the app into a container image  
6. Trivy scans the container images for vulnerabilities  
7. Kubernetes orchestrates the deployed containers  
8. AWS EC2 hosts the Kubernetes cluster infrastructure
